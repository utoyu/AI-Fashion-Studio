# 🕒 项目路线图与更新日志 (CHANGELOG.md)

## [2026-03-06] 电商组图核心重构与工作流打通
### ✨ 新特性 (New Features)
- **精英型录全面升级**：将原有的泛社交媒体文案生成器“精英型录”彻底重命名并重构为垂直领域的**“电商组图 (E-Commerce Catalog)”**。
- **动态电商矩阵**：构建了专业的主推图及副参考图组合，支持一键生成符合不同电商及内容平台要求的 16:9、3:4 等高分辨率营销长图。
- **本地+云端资产穿梭**：支持直接上传本地实拍图，新增 **[ 关联属性提取 ]** 创新功能——允许用户为本地图片独立挂接系统内标准资产的物理摘要（面料、洗护、工艺点等），而不会强行覆盖主图的视觉呈现。
- **全局词汇与导航打通**：统一主页、各类菜单中模块名称至“电商组图”，并将“素材库”导航层级提升至核心应用首位梯队（摄影室之后），强化素材底座地位。

## [2026-03-05] 智能素材库与 Agent 进化
### ✨ 新特性 (New Features)
- **多态详情系统**：为衣物、模特、背景分别设计了独立的 90vh 全屏详情编辑页，支持根据类型动态渲染属性（成分、角度图、生物学数据、环境光影等）。
- **角度图管理**：实现了衣物素材的主图、背图、侧图、顶/底图的批量上传、实时预览与管理功能。
- **商务资产包**：批量导入了 14 张高品质商务男装品牌相关资产，包含模特、外景办公场景及高支衬衫。

### 🛠️ 优化与修复 (Fixes & Optimizations)
- **摄影室工作流重组**：将混杂的功能块划分为“基础、主体、细节”三大语义化模块，降低了认知负荷。
- **Prompt 智能实验室**：引入了可视化标签云系统，支持“电影感、伦勃朗光”等专业提示词一键添加。
- **跨模块数据闭环**：打通了 `Photo Studio -> Smart Retouch` 的带参跳转链路，实现图片素材的秒级流转。
- **稳定性修复**：解决了 `Smart Retouch` 页面因 URL 异步传参导致的渲染崩溃与标签嵌套问题。
- **交互解耦**：解决了资产格子单/双击冲突问题，平衡了预览与编辑的操作逻辑。

## [2026-03-04] 文档标准化与工程规范
### ✨ 新特性
- **文档体系重构**：完成了从根目录到 `docs/` 的全量文档迁移，包含 PRD、架构说明、数据模型库及工作空间规则。
- **文件工程增强**：引入了 PowerShell 原生 `New-Item` 与 `Test-Path` 原子化操作，彻底解决了 Windows 环境下文件迁移时的路径占位错误。
- **工程资产标准化**：统一了全局 `Header` (h-20) 尺寸及导航层级。

## [2026-03-02] 核心 Demo 流程跑通
### ✨ 新特性
- **端到端流程验证**：实现了从“平铺图 -> AI 分析 -> 生成 Prompt -> 虚拟模特试穿图”的全链路 Mock 闭环。
- **多模型切换器**：引入了 Kimi 和 DeepSeek 的多模型选择器，支持根据响应质量动态切换提示词引擎。

## [2026-03-01] 需求规划与智能分发路由
### ✨ 新特性
- **业务线路定义**：在 PRD 中明确了“商务通勤”与“户外运动”两条核心业务线的 AI 生成策略差异。
- **智能分发层 (Backend)**：实现了后端 API 路由对 DeepSeek 和 Kimi 响应的统一清洗与 payload 转换，适配了复杂嵌套的 JSON 解析。
- **需求文档 (PRD) 1.0**：确立了 Phase 1 & 2 的研发里程碑，包含网络韧性设计与多模型路由规划。
